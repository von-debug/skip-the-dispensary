# Skip the Dispensary

## Project Description
This project is a web application designed to provide users with a seamless experience in finding and purchasing cannabis products. It aims to simplify the process of ordering from dispensaries and offers information about various products.

## Features
- User-friendly interface
- Search and filter products
- Product reviews and ratings
- Secure payment processing
- Ability to track orders

## Instructions for Running the Website
1. Clone the repository:
   ```bash
   git clone https://github.com/von-debug/skip-the-dispensary.git
   ```
2. Navigate to the cloned directory:
   ```bash
   cd skip-the-dispensary
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Visit `http://localhost:3000` in your browser to access the application.
